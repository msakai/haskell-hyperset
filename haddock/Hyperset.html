Accessible Graph.
type Tagging u = FiniteMap Vertex u
Let G be an accessible graph, and let U={φ}∪u be a collection containing the empty set and urelements. A tagging of G is a function t: G->U that assigns to each childless node of G an element of U. And we interpret lookupFM t n == Nothing for a childless node n of G as t(v)=φ.
type Decoration u = Table (UrelemOrSet u)

A decoration is a function d defined on each node n of G such that

  • d(n) = t(n) (if n is childless)
  • d(n) = {d(m) | m∈children(n)}
